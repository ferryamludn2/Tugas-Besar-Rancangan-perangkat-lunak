Scenario
Steps
1. menakan tombol update ketersediaan kursi pesawat yang dipilih
2. SYSTEM menampilkan tampilan untuk menaikkan atau mengurangi kursi yang tersedia pada pesawat
3. if jika menakan tombol "+"
    3.1. SYSTEM menambahkan kuantitas kursi yang kosong untuk sementara
4. else if jika menekan tombol "-"
    4.1. SYSTEM mengurangi kuantitas kursi yang kosong untuk sementara
   end if
5. menekan tombol update
6. SYSTEM menampilkan prompt "Apakah anda yakin?"
7. if jika "Ya"
    7.1. SYSTEM menambahkan atau mengurangi kuantitas kursi yang kosong ke sistem secara permanen
8. else if jika "Tidak"
    8.1. SYSTEM mengembalikan ke halaman update ketersediaan kursi
   end if
